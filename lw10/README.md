# Harmonics Chart Drawer (Electron + TypeScript, MVVM)

**По заданию:**
- Десктопное приложение.
- Паттерн: **MVVM** (можно считать как вариант из MVC/MVP/MVVM).
- Пользователь может **добавлять/удалять** гармоники.
- Пользователь может **редактировать** амплитуду/частоту/фазу и выбирать **sin/cos**.
- **График суммы** гармоник **обновляется автоматически** при любых изменениях.
- Бонус: вкладки **Chart / Table** (представление график/таблица).
- Бонус: **юнит‑тесты** для модели.

## Быстрый старт

1) Установить зависимости:
```bash
npm i
```

2) Запуск:
```bash
npm run start
```

3) Тесты:
```bash
npm run test
```

## Структура

- `src/core/*` — **Model** и математика (можно тестировать отдельно).
- `src/core/HarmonicsViewModel.ts` — **ViewModel** (команды + события).
- `src/renderer/*` — **View** (DOM + Canvas график + таблица).
- `src/main/main.ts` — Electron main process.

## Формула гармоники

Каждая гармоника:

`A * sin( f * x + p )`  или  `A * cos( f * x + p )`

График строится для суммы всех гармоник.
По умолчанию строим точки на отрезке `x = 0..5` с шагом `0.05` (как на примере из задания).
